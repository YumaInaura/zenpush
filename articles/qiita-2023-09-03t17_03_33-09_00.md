# 例

mainブランチの中で2個のコミットを選んでRevertしたい場合

## gitでの操作

```
git switch main
git checkout -b <てきとうなブランチ名>

git revert (1個目のコミットハッシュ) (マージコミットではなく元のコミット)
git revert (2個目のコミットハッシュ) (マージコミットではなく元のコミット)

git push
```



## Githubでの操作

`main < <てきとうなブランチ名>` のPRを作成
